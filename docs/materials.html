🔧 Инструкция по работе с GPU в Colab

  1. Откройте ноутбук в Google Colab (кнопка "🚀 Colab" выше)
  2. В меню выберите "Runtime" → "Change runtime type"
  3. В поле "Hardware accelerator" выберите "GPU"
  4. Нажмите "Save"
  5. Теперь алгоритмы PD3 и PALMAD будут работать на GPU
