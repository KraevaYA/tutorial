Варианты работы с материалами

Работать с материалами можно двумя способами:

1. В облачной среде Google Colab
Не требует локальной установки Python и ключевых библиотек. Вычисления выполняются на удаленных серверах Google с доступом к GPU (с ограничениями по времени). Для работы достаточно браузера и учетной записи Google.
📋 Показать инструкцию ▼
Шаг 1. Откройте ноутбук с заданиями в Google Colab по ссылке:
Открыть ноутбук в Colab
Шаг 2. Настройте среду выполнения с GPU. В открывшемся ноутбуке выберите в меню Runtime → Change runtime type. В появившемся окне в поле Hardware accelerator выберите GPU и нажмите Save.
Шаг 3. Выберите вариант работы (временный или с сохранением на Google Drive), скопируйте соответствующий код в первую ячейку ноутбука и выполните ее:
🔹 Вариант А. Временная сессия (без сохранения)
# Клонирование репозитория с материалами
!git clone https://github.com/username/pavt-tutorial-2026.git
%cd pavt-tutorial-2026

# Установка зависимостей
!pip install -r requirements.txt -q
🔹 Вариант Б. С хранением на Google Drive (сохранение прогресса)
from google.colab import drive
drive.mount('/content/drive')

!git clone https://github.com/username/pavt-tutorial-2026.git /content/drive/MyDrive/pavt-tutorial
%cd /content/drive/MyDrive/pavt-tutorial

!pip install -r requirements.txt -q
2. На локальном компьютере
Требует скачивания архива с материалами на странице сайта или клонирования GitHub-репозитория, видеокарту NVIDIA с поддержкой CUDA и установленные драйверы.
📋 Показать инструкцию ▼
Шаг 1. Получите материалы одним из способов:
🔹 Вариант А. Клонирование репозитория (через Git) с помощью команды в терминале:
git clone https://github.com/username/pavt-tutorial-2026.git
🔹 Вариант Б. Скачивание ZIP-архива и его распаковка:
  • на странице сайта (раздел «Доступ к материалам», кнопка «Скачать ZIP»);
  • на странице репозитория GitHub: кнопка Code → Download ZIP.
Шаг 2. Перейдите в папку проекта и настройте окружение, выполнив команды в терминале: 
cd pavt-tutorial-2026             # переход в папку проекта
python -m venv venv                # создание виртуального окружения
source venv/bin/activate           # активация окружения (Linux/macOS)
# venv\Scripts\activate            # активация окружения (Windows)
pip install -r requirements.txt    # установка зависимостей
jupyter notebook                   # запуск Jupyter Notebook
В открывшемся интерфейсе Jupyter выберите notebooks/pavt-tutorial-2026.ipynb.

✅ После выполнения всех шагов можно приступать к выполнению заданий. При возникновении затруднений используйте готовое решение.
